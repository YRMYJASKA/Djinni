# Djinni
Terminal text editor for Linux

# Usage
``` djinni <filename | --version | --help> ```
# Installation
## Prerequisites
* ``` git ```
* ``` clang++ 5.0.1 ```
* ``` ncurses ```, a C library 
* ``` GNU make ```
## Process
1. Download the git repository and cd into it. 

``` git clone https://github.com/YRMYJASKA/Djinni ```

  and

```  cd Djinni```



2. To build Djinni. The executable will be in 'bin/'

``` make install && make ```

3. To launch djinni.

``` bin/djinni ```
